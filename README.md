📊 Product Sales 2023–2024 Dashboard

This project presents an interactive Sales Analytics Dashboard built to analyze product performance, revenue, and profitability across different regions, cities, and product categories for 2023–2024.
The dashboard provides clear business insights through key performance indicators and visual analytics.
